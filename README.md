# Fresh project

Your new Fresh project is ready to go. You can follow the Fresh "Getting
Started" guide here: https://fresh.deno.dev/docs/getting-started

### Usage

Make sure to install Deno:
https://docs.deno.com/runtime/getting_started/installation

Then start the project in development mode:

```
deno task dev
```

This will watch the project directory and restart as necessary.

I live in a large coliving house and startup company where we all live and work
together and sometimes do short rental of rooms for digital nomads in the tech
field, and it's pretty great but we still have several issues that need to be
solved:

- Split bills fairly and easily
- Split chores fairly and easily
- Chat with each other, but in a way that doesnt get lost in the sauce, like a
  chat board or mural
- Mood tracker for each housemate (so we can see if someone needs some space, or
  just a pick me up)
- Daily ice breaker question to facilitate bonding
- Gamification system that rewards good behavior (doing chores, helping each
  other, etc.)
- Manage house members (add, remove, etc.)
- Create "events" to invite housemates to join for fun activities
- Create groups for sub-residents (e.g. full time vs short term)

Maybe we can use ai to create avatars for each housemate. and maybe an ai that
can analyze the house and suggest activities to do together.

A gamified coliving web app to split household chores and bills, with a mural
for communication between residents, a mood tracker and a daily ice breaker
question so people get to know each other better
